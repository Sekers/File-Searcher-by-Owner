File Searcher by Owner
 

File Searcher by Owner is the number one utility for finding files and folders owned by a particular user or group.

 

Download

 

Features
Search by a local or domain user or group
Search by Security Identifier (SID)
Supports UNC paths
Export to comma-separated (CSV) file
Easily change ownership of found files
Preserves right-click context menu for files & folders
Portable and lightweight
 

 
System Requirements
Windows 7 or higher, Windows Server 2008 SP2 or higher (both 32-bit and 64-bit)
.NET Framework 4.1 or higher (included in Windows 8.1/Server 2012 R2 and higher; otherwise needs to be installed manually)

For older operating systems (Windows XP/Vista/Server 2003), download final .NET 2.0 version
